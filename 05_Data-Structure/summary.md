# Data Structure

1. Array : Definisi, karakteristik, deklarasi, inisialisasi nilai array, length/panjang array, serta iterasi array
2. Slice : Definisi, karakteristik, length dan capacity dari slice, inisialisasi dan deklarasi array, append() and  copy(), zero value dari slice
3. Map : Definisi, karakteristik, key & value dan penggunaan map dalam bahasa pemrograman GO
4. Function : Definisi, deklarasi function, function dengan dan tanpa parameter, return value function (single & multiple), named return parameter
